# scTagging

[![medRxiv](https://img.shields.io/badge/medRxiv-blue?style=flat)](https://www.medrxiv.org/content/10.64898/2026.02.15.26346353v1)

This repo contains code for implementing analyses in Dorans et al. medRxiv, "Distinguishing causal from tagging enhancers using single-cell multiome data." See below (and analysis-specific folders) for steps to compute ATAC peak-level and gene-level scores, implement stratified co-accessibility score regression, and fine-map ATAC peaks to genes.

Clone the repository using:

```bash
git clone https://github.com/elizabethdorans/scTagging.git
cd scTagging
```

Create the conda environment using:

```bash
conda env create -f environment.yaml
conda activate sctagging
```

Submit jobs to your own remote cluster (e.g. using [sbatch](https://slurm.schedmd.com/sbatch.html)):

```bash
cmd="<command>"; sbatch --time=<time> --mem=<mem> ... --wrap="$cmd"
```

Approximate memory and time requirements are given for computationally intensive tasks, but these will need to be adjusted for different data sets.

## Example data

This repository includes a small example data set at:

```text
example_data/
```

The example folder contains a selection of data from the Xu K562 data set generated by Steps 0-1 below: peak-gene correlations and peak-peak correlations (co-accessibility) from chromosome 22 only, as well as RNA and ATAC matrices (`RNA_metacell_matrix.rds` and `ATAC_metacell_matrix.rds`). These example files are intended for running the downstream score and fine-mapping commands in Steps 2-4.

Example workflow output can be written to:

```text
example_outputs/
```

## Step 0: Create an ArchR project

Follow the instructions in the [E2G method tutorials](https://github.com/elizabethdorans/E2G_Method_Tutorials) to save single-cell multiome data into an ArchR project.

## Step 1: Compute peak-peak co-accessibility and peak-gene links using ArchR

You can run this code using any single-cell RNA+ATAC-seq multiome data set; see Data Availability and Supplementary Table 1 in Dorans and Price medRxiv for examples and download links.

The script `run_archr_peak_gene_linking_coaccessibility_metacells.R` takes as input an ArchR project for a single-cell RNA+ATAC multiome data set, computes peak-peak correlations for all peak-peak pairs with distance <1Mb, computes peak-gene correlations for all peak-gene pairs with distance <1Mb, and saves RNA and ATAC metacell-level matrices for downstream analyses.

Example command: [~1 hour, ~20G]

```bash
Rscript run_archr_peak_gene_linking_coaccessibility_metacells.R \
  --archr_proj_dir <archr_proj_dir> \
  --out_dir <out_dir>
```

`<archr_proj_dir>`: Path to a folder containing an ArchR project with ATAC and RNA data from a single-cell RNA+ATAC multiome data set (same cell barcodes in both RNA and ATAC matrices).\
`<out_dir>`: Path to a folder where outputs will be saved (peak-peak co-accessibilities, peak-gene links, and RNA and ATAC metacell-level matrices).\
`<distance_threshold>`: [OPTIONAL] Maximum peak-peak and peak-gene distance for computing peak-peak and peak-gene correlations (default is 1Mb).\
`<threads>`: [OPTIONAL] Number of threads for ArchR operations (default is 1).

Outputs:

1. Peak-peak correlations at `<out_dir>/coaccessibility_dist<distance_threshold>.tsv`
2. Peak-gene correlations at `<out_dir>/peak_gene_links_dist<distance_threshold>.tsv`
3. RNA metacell matrix at `<out_dir>/RNA_metacell_matrices/RNA_metacell_matrix.rds`
4. ATAC metacell matrix at `<out_dir>/atac_metacell_matrices/ATAC_metacell_matrix.rds`

## Step 2a: Compute peak scores

The `peak_scores/` folder contains code for computing co-accessibility and co-activity scores for each ATAC peak in a single-cell RNA+ATAC multiome data set.

See [peak_scores/README.md](peak_scores/README.md) for the full workflow. The example inputs included in this repository are:

```text
example_data/coaccessibility_dist1000000_chrom22.tsv
example_data/peak_gene_links_dist1000000_chrom22.tsv
example_data/RNA_metacell_matrix.rds
example_data/ATAC_metacell_matrix.rds
```

Example commands using the included example data:

```bash
Rscript get_background_coaccessibility_via_metacell_downsampling.R \
  --norm_atac_file example_data/ATAC_metacell_matrix.rds \
  --num_peak_pairs 100000 \
  --outfile example_outputs/background_coaccessibility.tsv

Rscript peak_scores/bias_corrected_coaccessibility_scores.R \
  --uncorrected_file example_data/coaccessibility_dist1000000_chrom22.tsv \
  --background_coacc_file example_outputs/background_coaccessibility.tsv \
  --outfile example_outputs/coaccessibility_scores.tsv

Rscript get_background_coactivity_via_metacell_downsampling.R \
  --norm_atac_file example_data/ATAC_metacell_matrix.rds \
  --norm_rna_file example_data/RNA_metacell_matrix.rds \
  --gene_universe_file gene_TSS.txt \
  --num_peak_gene_pairs 100000 \
  --outfile example_outputs/background_coactivity.tsv

Rscript bias_corrected_coactivity_scores.R \
  --uncorrected_file example_data/peak_gene_links_dist1000000_chrom22.tsv \
  --background_coacc_file example_outputs/background_coactivity.tsv \
  --gene_universe_file gene_TSS.txt \
  --outfile example_outputs/coactivity_scores.tsv \
  --focal_feature peak
```

## Step 2b (optional): Compute gene scores

The `gene_scores/` folder contains code for computing gene co-expression and gene co-activity scores for each gene in a single-cell RNA+ATAC multiome data set.

See [gene_scores/README.md](gene_scores/README.md) for the full workflow. The file `gene_TSS.txt` in the main folder provides the default gene universe used by the gene-score scripts.

Example commands using the included example data:

```bash
python gene_scores/generate_proximal_gene_gene_pairs.py \
  --rna_matrix_file example_data/RNA_metacell_matrix.rds \
  --gene_universe_file gene_TSS.txt \
  --outfile example_outputs/gene_scores/proximal_gene_gene_pairs.tsv

Rscript gene_scores/gene_gene_coexpression.R \
  --rna_matrix example_data/RNA_metacell_matrix.rds \
  --gene_gene_pairs_file example_outputs/gene_scores/proximal_gene_gene_pairs.tsv \
  --outfile example_outputs/gene_scores/gene_gene_coexpression.tsv

Rscript get_background_coaccessibility_via_metacell_downsampling.R \
  --norm_atac_file example_data/RNA_metacell_matrix.rds \
  --num_peak_pairs 100000 \
  --outfile example_outputs/gene_scores/background_coexpression.tsv
  
Rscript bias_corrected_coactivity_scores.R \
  --uncorrected_file example_data/peak_gene_links_dist1000000_chrom22.tsv \
  --background_coacc_file example_outputs/background_coactivity.tsv \
  --gene_universe_file gene_TSS.txt \
  --outfile example_outputs/gene_scores/coactivity_scores.tsv \
  --focal_feature gene
```

## Step 3: Run S-CASC

The `S-CASC/` folder contains code to implement stratified co-accessibility score regression.

See [S-CASC/README.md](S-CASC/README.md) for the full workflow.

## Step 4: Run peak-gene fine-mapping

The `peak_gene_finemapping/` folder contains code to implement peak-gene fine-mapping.

See [peak_gene_finemapping/README.md](peak_gene_finemapping/README.md) for the full workflow.

Example command using the included example data:

```bash
Rscript peak_gene_finemapping/peak_gene_finemapping.R \
  --coactivity_file example_data/peak_gene_links_dist1000000_chrom22.tsv \
  --rna_file example_data/RNA_metacell_matrix.rds \
  --atac_file example_data/ATAC_metacell_matrix.rds \
  --outfile example_outputs/peak_gene_finemapping/pips.tsv
```
