# Example data

This folder contains chromosome 22 example inputs generated from the Xu K562 data set:

```text
ATAC_metacell_matrix.rds
RNA_metacell_matrix.rds
coaccessibility_dist1000000_chrom22.tsv
peak_gene_links_dist1000000_chrom22.tsv
```

These files are used by the example commands in the main README for peak scores, gene scores, and peak-gene fine-mapping.
